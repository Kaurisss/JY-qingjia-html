## 动态请假条页面

这是一个用于展示和自定义学生请假信息的静态HTML页面。它使用本地存储（localStorage）来动态填充和修改页面上的所有数据，包括个人信息、请假详情和审批状态。

页面内置了一个隐藏的编辑功能，允许用户实时修改所有可见内容。

### 主要功能

- **数据动态加载**：页面加载时，所有信息（如姓名、学号、请假事由、时间等）均从浏览器的 `localStorage` 中读取。
- **内容高度可定制**：内置一个隐藏的编辑模态框，允许用户修改页面上的每一个字段。
- **头像更换**：编辑功能支持上传并更换新的头像图片。
- **本地持久化**：所有修改都保存在 `localStorage` 中，因此在刷新页面后更改依然有效。
- **响应式布局**：为移动设备优化，模拟了移动应用中的请假审批界面。

### 如何使用

1.  **查看页面**：
    直接在现代浏览器中打开 `index.html` 文件。

2.  **激活编辑模式**：
    - 连续快速双击页面顶部的“请假”标题栏。
    - 页面右下角将出现一个蓝色的“编辑”按钮。

3.  **修改信息**：
    - 点击“编辑”按钮，会弹出一个包含所有可编辑项的表单。
    - 在此表单中，您可以修改姓名、学号、请假事由、目的地、时间，甚至上传新的头像。

4.  **保存更改**：
    - 填写完毕后，点击表单底部的“保存”按钮。
    - 页面将会自动刷新，并显示您刚刚输入的新信息。所有更改都将保存在您的浏览器中。
